## Visual Identity Summary
糖果炸裂风——Q版三头身角色在暗色地牢中释放糖果色爆裂技能。一切遵循"软萌外壳+硬核内芯"原则：角色和UI是圆润滑稽的，但技能特效和打击反馈是尖锐炸裂的。暗底亮效，关掉特效画面变暗沉，打开特效画面如烟花炸开。

## Art Style

### Rendering Style
2D手绘精灵帧动画，Godot 2D渲染管线。角色和场景均为预渲染2D素材，不使用3D模型。特效使用 Godot GPUParticles2D + 着色器后处理。

### Proportions
- **角色**: 三头身Q版（头:身:腿 = 1:1:1），头部占比极大，四肢短粗
- **场景**: 横版2.5D纵深（3-4层视差滚动），角色活动层在中间层
- **UI**: 大字体、圆角矩形、图标优先，适配1280x720基础分辨率
- **比例对比**: 角色高度约占屏幕1/6，Boss占1/3~1/2

### Level of Detail
- **角色**: 中高细节——帧动画8-16帧/动作，关键帧挤压拉伸
- **场景**: 中等细节——分层tileset，前景装饰+中景交互层+远景氛围
- **UI**: 低细节但高辨识——大色块+图标，减少装饰
- **特效**: 高细节——粒子系统+着色器，战斗反馈是核心卖点

### Visual Hierarchy
1. **永远最突出**: 玩家角色 + 敌人 + 伤害数字（高饱和、高对比）
2. **次要突出**: 技能特效区域（闪光但透明度可控）
3. **背景层**: 场景环境（低饱和、低对比、不干扰战斗阅读）
4. **最底层**: HUD/UI（固定位置、半透明底色、不遮挡战斗区）

## Character Art Standards

### 基本规范
- 三头身比例，头大身短腿更短
- 粗线条轮廓（2-3px黑色描边），确保在任何背景上可读
- 每个职业有唯一主色（蓝/红/绿/紫等），轮廓+描边一致
- 眼睛比例夸张（占头部1/3），表情系统2-3种（正常/愤怒/痛苦）

### 动画规范
- 站立idle: 8帧，呼吸式微动
- 走路: 8帧，弹性步伐
- 跑步: 6帧，Q版夸张摆臂
- 轻攻×3: 每段4-6帧，快速斩击
- 重攻: 8帧，蓄力→释放两段
- 必杀: 12-16帧，蓄力光效→爆发→收招
- 受击: 4帧，挤压变形→弹回
- 跳跃: 4帧，起跳→滞空→下落

### 变形规则（打击感核心）
- 攻击蓄力: 横向压缩0.7x + 纵向拉伸1.3x
- 命中瞬间: 横向拉伸1.3x + 纵向压缩0.7x
- 必杀释放: 极端拉伸1.5x + 后续弹回0.8x→1.0x
- 受击: 压缩0.6x + 弹回1.1x→1.0x

## Environment Art Standards

### 场景分层（从后到前）
1. **远景层** (Far): 天空/远山/城池剪影，视差0.1x
2. **中远层** (Mid-Far): 建筑/大树/城堡轮廓，视差0.3x
3. **中景层** (Mid): 可交互平台、门、装饰物，视差1.0x（角色层）
4. **近景层** (Near): 前景装饰柱/帘幕，视差1.5x，半透明

## UI Art Standards

### HUD布局
- 左上: 角色头像 + HP条（圆角矩形，绿→黄→红渐变）
- 左上下方: MP/能量条
- 右上: 副本进度（房间x/N）
- 中上: 连击计数器（COMBO x N），大字体，命中时放大脉冲
- 右下: 技能栏（4格，J/K/L/空格图标）

### 设计原则
- 圆角矩形为主（radius ≥ 8px），与Q版风格一致
- 半透明底色（alpha 0.6-0.8），不遮挡战斗
- 图标优先于文字，技能图标必须颜色编码
- 伤害数字必须可读：3px黑色描边 + 高饱和前景色
- 最小字体: 18px（1080p基准），标题24px

### 字体选择
- 中文: 圆体/黑体（无衬线，匹配Q版气质）
- 数字/英文: 像素风或粗圆体（DAMAGE NUMBER用粗体）
- 禁用: 衬线体、手写体（与Q版风格冲突）

## VFX Standards

### 粒子风格
- 圆形/星形粒子为主，不用写实烟雾
- 颜色饱和度极高（saturate > 80%）
- 生命周期短（0.2-0.5秒），强调瞬间爆发
- 大量使用"缩放+淡出"而非"缓慢消散"

### 打击反馈特效
| 类型 | 效果 | 颜色 |
| ---- | ---- | ---- |
| 轻攻命中 | 小型圆弧斩击线 + 3-5粒子 | 白色#FFFFFF |
| 重攻命中 | 大型弧形斩击 + 10-15粒子 + 闪光 | 黄色#FFD700 |
| 必杀命中 | 全屏闪白 + 冲击波环 + 30+粒子 + 速度线 | 红#FF4444 + 金#FFD700 |
| 暴击 | 额外星形爆炸 + 屏幕短暂金色叠层 | 金色#FFD700 |

### 屏幕后处理
- 命中闪白: 全屏白色叠加0.05秒
- 必杀慢动作: 时间缩放0.2x持续0.3秒
- 低血量: 屏幕边缘红色渐晕（vignette）
- 连击高combo: 屏幕轻微色差偏移（chromatic aberration）

## Asset Production Standards

### Naming Convention
`[category]_[name]_[variant]_[size].[ext]`

示例:
- `char_warrior_idle_64x64.png`
- `env_cave_tile_64x64.png`
- `vfx_slash_heavy_128x128.png`
- `ui_btn_skill_48x48.png`

### Texture Standards
| Category | Max Resolution | Format | Color Space |
| -------- | -------------- | ------ | ----------- |
| Characters | 256x256 per sprite sheet | PNG | sRGB |
| Environments | 512x512 per tileset | PNG | sRGB |
| UI | 256x256 per element | PNG | sRGB |
| VFX | 256x256 per effect sheet | PNG | sRGB (预乘alpha) |

### Animation Standards
- 帧率: 24fps导出，Godot中按需降速
- blend time: 普通动作0.1s，战斗动作0.05s（快切换）
- 动画图: 分层（Base=身体, Overlay=特效, Additive=表情）
- 循环动画首尾帧必须一致

## Accessibility
- Colorblind-safe UI elements required — 暴击/必杀除颜色外必须有形状标识（星形/爆炸形）
- Minimum text size: 18px at 1080p
- High contrast mode: 暗背景上所有文字/数字强制3px描边
- Icon + color (never color alone) for game state — HP颜色渐变必须同时显示数字
- 伤害数字默认启用，提供"简化特效"选项减少屏幕干扰
