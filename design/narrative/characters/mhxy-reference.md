## 梦幻西游Q版设计通则（适用于格斗萌主）

### 1. 比例与轮廓
- **头身比**: 2-2.5头身（我们当前3头身偏大，可考虑缩小到2.5）
- **轮廓**: 每个角色有独特的外轮廓剪影，即使缩到32px也能辨认
- **武器**: 武器比身体大（1.2-1.5倍），是角色辨识度的关键锚点

### 2. 发型与头部
- **不要完全光头** — 即使是武僧，也应有短发或束发，顶部可以有发髻
- **刘海是标配** — 几乎所有角色都有刘海，增加萌感
- **发饰增加辨识度** — 发带、发簪、发冠、兽耳等
- **眼睛**: 大但有细节（瞳孔+高光+虹膜颜色），不要纯黑点

### 3. 服饰
- **交领/对襟** — 中国传统服装结构，V形领口
- **腰带+飘带** — 几乎所有角色都有腰带和飘带元素，增加动感
- **护甲装饰** — 肩甲、护腕、护膝，但不覆盖全身（保留布衣感）
- **云纹/回纹/兽面纹** — 中国传统纹样点缀

### 4. 武器/法器
- **oversized** — 比角色本身还大，形成反差萌
- **有装饰** — 剑穗、流苏、符文、宝石
- **有颜色编码** — 武器颜色和角色主色一致或互补

### 5. 配色
- **主色+辅色+点缀色** 三色系
- **主色** 占60%，**辅色** 占30%，**点缀色** 占10%
- **避免纯黑纯白** — 用深灰/米白代替
- **中国传统色** — 朱砂红、石青蓝、藤黄、黛紫

### 6. 表情
- **大眼+小嘴** — Q版标配
- **腮红** — 几乎所有女性角色和萌系男性角色都有
- **眉毛传达性格** — 上扬=热血，平直=憨厚，弯曲=温柔

## 对我们三个角色的改进建议

###  慧空 → 参照 剑侠客 + 化生寺僧人

| 当前设计 | 梦幻西游参考 | 建议改进 |
|----------|-------------|----------|
| 完全光头 | 束发/短发+发髻 | 改为**短寸头+小发髻**（武僧头），保留一些碎发刘海 |
| 简单红头带 | 多层发带+飘带 | 保留红头带，增加**额饰**（金色小圆片） |
| 拳头缠绑带 | 拳套/护手 | 改为**金色护手**（更像武器），增加流苏 |
| 念珠在脖子上 | 念珠手串 | 念珠改为**手持念珠**（更像武器/法器），脖子上保留少量 |
| 赤脚 | 草鞋/布鞋 | 改为**武僧布鞋**（白色/灰色） |
| 简单武服 | 劲装+云纹 | 增加**云纹暗纹**，衣摆有渐变 |
| 无披风 | 披风/袈裟 | 增加**半透明袈裟**（金色薄纱质感） |

### 🍬 糖糖 → 参照 狐美人 + 龙宫仙子

| 当前设计 | 梦幻西游参考 | 建议改进 |
|----------|-------------|----------|
| 道冠（方帽） | 发髻+发饰 | 改为**双环望仙髻**（经典仙侠发型），配金色发簪 |
| 围裙 | 唐风裙装 | 改为**粉色齐胸襦裙**（唐风），腰间铃铛 |
| 白围裙 | 披帛 | 增加**半透明披帛**（薄荷绿，飘逸感） |
| 葫芦杖 | 拂尘/法杖 | 保留葫芦，但改为**拂尘法杖**（尾端是葫芦+流苏） |
| 靴子 | 绣鞋 | 改为**翘头绣花鞋**（粉色+金线） |
| 简单手臂 | 水袖 | 增加**水袖**（宽袖口，飘动感） |
| 腮红 | 已有 | 保持，增加**花钿**（额头装饰） |

###  小鬼丸 → 参照 巨魔王 + 狮驼岭

| 当前设计 | 梦幻西游参考 | 建议改进 |
|----------|-------------|----------|
| 短小角 | 粗壮牛角 | 改为**粗壮弯角**（向后弯，更有气势） |
| 简单肚兜 | 铠甲+战裙 | 肚兜保留，但增加**肩甲**（兽面纹，更大更威武） |
| 锯齿嘴 | 獠牙 | 保留锯齿嘴，增加**两颗小獠牙**（从嘴角露出） |
| 菱形锤 | 狼牙棒/巨斧 | 改为**狼牙棒**（国风降魔杵+狼牙元素） |
| 简单拳头 | 带铆钉的拳套 | 拳头增加**铁质指虎** |
| 铃铛 | 骨饰/骷髅饰 | 保留铃铛，增加**小骷髅挂饰** |
| 无尾巴 | 小尾巴 | 增加**小恶魔尾巴**（末端有火焰/心形） |
