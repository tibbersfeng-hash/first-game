# Character: 萌鬼狂战士 · 小鬼丸

## Quick Reference

- **Full Name**: 小鬼丸（Kiguemaru）
- **职业代号**: Oni Crushkin
- **Role in Story**: 主角备选 — 从地狱逃出来的小妖怪，在人间寻找"家"的感觉
- **Role in Gameplay**: 可玩主角职业 #3 — 近战爆发型
- **First Appearance**: 副本选择画面（可选角色）
- **Status**: Provisional

---

## Concept

小鬼丸是一个从熔岩地狱逃出来的小鬼族，头上长着一对可爱的角，扛着一把比自己大三倍的巨型糖果锤。看起来软萌无害，但每一锤落地都能让大地龟裂。他/她在人间游荡，用暴力方式交朋友——"跟我打一架就是朋友了！"

玩家应该喜欢小鬼丸的原因：**暴力萌的极致**——用最可爱的外表打出最恐怖的伤害。每次挥锤都是地震，每次砸中都是屏幕炸裂。简单、粗暴、爽快。

## Appearance

### Build
- 7头身正常比例，身高约165cm，矮壮结实体型
- 体型结实，肩宽腰粗，比其他两个角色宽20%
- 鬼角让整体高度增加，整体感觉是"壮实的鬼族少年"

### Distinguishing Features
- **鬼角（一对）** — 头顶两只短短的小角，红色，蓄怒时角尖发光
- **巨型糖果锤** — 锤子头是彩色糖果造型，但明显是实心的（重得要命）
- **红色肚兜** — 日式红色肚兜+肩甲（铁制），下半身是短裤
- **锯齿状嘴巴** — 微笑时露出一排小尖牙，看起来凶但其实很萌

## Personality

### Core Traits
- **单纯直率** — 喜怒哀乐全写在脸上，不会伪装
- **好战但友善** — 见面先打一锤，打完再问名字
- **食量惊人** — 什么都能吃，包括敌人的武器（咀嚼声很响）
- **怕寂寞** — 一个人在城镇里会到处找人玩（打架）

### Voice Profile
- **Speech Pattern**: 短句、重复、简单词汇。兴奋时重复同一句话
- **Vocabulary Level**: 非常简单，偶尔说出让人意外的哲理句
- **Verbal Tics**: "嘿嘿！" / "砸！" / "好玩好玩！"
- **Tone Reference**: 小孩子的天真 + 怪兽的力量 + 偶尔的深沉

### Emotional Range

| Emotion | Trigger | Expression | Example Line |
|---------|---------|-----------|-------------|
| 开心 | 打架/吃东西 | 锯齿嘴大张，角轻微发光 | "嘿嘿！好玩好玩！再来！" |
| 愤怒 | 朋友被欺负 | 眼睛变红，角亮红光，身体变大10% | "不许……欺负……朋友！" |
| 困惑 | 复杂的问题 | 歪头+挠头+锤柄戳地 | "嗯……不懂……砸了再说？" |
| 难过 | 想起地狱 | 角的光暗下来，锤头垂地 | "……想回家……但是回不去了……" |
| 满足 | 吃饱+打赢 | 躺在地上，锤放肚子上 | "嘿嘿……舒服……" |

## Motivation and Arc

### Primary Motivation
寻找"家"——小鬼丸从熔岩地狱逃出来，在人间流浪。他/她战斗不是为了变强，而是为了找到愿意接纳自己的地方和人。

### Character Arc (Lightweight)

| Phase | State | Trigger |
|-------|-------|---------|
| 初始 | 流浪小妖怪，到处找人打架交朋友 | 角色选择 |
| 成长 | 在冒险中遇到真正的伙伴 | 和武僧/魔导师组队 |
| 蜕变 | 面对是否回到地狱的选择 | 通关熔岩地狱 |
| 终局 | "有朋友的地方就是家" | 通关最终副本 |

### Internal Conflict
小鬼丸体内有鬼族的狂暴血脉——打得太嗨会失控。他/她需要学会在"释放力量"和"保持自我"之间找到平衡。这个矛盾在鬼化变身机制中体现。

## Gameplay Function

### 职业定位
**近战爆发型** — 慢速高伤，挨打攒怒，一击制霸。

### 核心机制：怒气值

```
怒气值（Rage）：0-100
- 被击中+8怒气（全职业最高受击回复）
- 攻击命中+3怒气
- 重攻消耗15怒气（但怒气>50时免费）
- 必杀消耗50怒气
- 怒气>80时进入"半鬼化"：攻击力+20%，受击减伤15%
- 怒气=100时释放必杀后进入" fully 鬼化"状态5秒：全属性+30%

鬼化视觉反馈：
- >80: 角发红光，眼睛从金色变橙色，皮肤紫色加深
- =100: 角全亮，眼睛血红，身体微微放大，暗紫气焰环绕
```

### 攻击风格

| 攻击 | 动作描述 | 特效 | 手感 |
|------|----------|------|------|
| 轻攻1 | 单手横扫锤（慢但范围大） | 暗紫弧形冲击 | 沉重横扫 |
| 轻攻2 | 锤柄突刺 | 紫色突刺线 | 直线穿刺 |
| 轻攻3 | 双手过顶砸地 | 地面裂缝+震荡波（AOE） | 地动山摇 |
| 重攻 | 跳砸（跳起→锤击地面） | 大范围圆形冲击波 | 陨石撞击 |
| 必杀 | 鬼王降临·地狱碎灭锤 | 鬼化全身→巨锤变大2倍→连续砸地7次→最终一击 | 毁灭级打击 |
| 闪避 | 翻滚（带着锤子笨重但快） | 紫色残影+地面刮痕 | 沉重翻滚 |

### 连招路径

```
基础连招（正面硬刚）:
J → J → J（横扫→突刺→砸地）→ 等CD → 再来一遍

进阶连招（怒气管理）:
J-J-J（攒怒）→ 被敌人打一下（+8怒）→ K（跳砸，怒气>50免费）→ 继续

高手连招（鬼化爆发）:
故意吃几下攻击攒满怒 → L必杀（鬼化状态）→ 5秒鬼化内 J-J-K-J-J-K 疯狂输出
```

### What This Character Provides
- **最高单次伤害** — 每一锤都是"重击"，数字最大
- **最强容错** — 挨打反而更强，新手不会轻易死
- **最广AOE** — 攻击范围全职业最大，清怪效率最高
- **风险/回报低** — 简单直接，适合"不想思考只想爽"的玩家
- **视觉冲击** — 每一锤都是屏幕震动+地裂特效，打击感最炸

## Animation Feel

### Idle（8帧循环）
1. 扛着锤子左右晃（锤柄架肩上）
2-3. 突然把锤放下来→好奇地看着锤子上的糖果纹
4-5. 甩甩手臂→锤柄戳地
6. 角闪了一下光→摸摸角
7-8. 重新扛起锤子，蹦蹦跳

### Attack Deformation Rules
| 动作 | 横向缩放 | 纵向缩放 | 持续时间 |
|------|----------|----------|----------|
| 轻攻横扫 | 1.3x | 0.8x | 5帧（慢） |
| 轻攻砸地 | 1.0x | 0.6x → 弹1.2x | 6帧 |
| 重攻跳砸 | 0.7x（蓄力）→ 1.5x（砸地） | 1.4x → 0.6x | 4+6帧 |
| 必杀蓄力 | 身体放大1.2x | 角光芒扩散 | 10帧 |
| 必杀连续砸 | 每砸一次放大5% | 极端挤压 | 持续 |
| 受击 | 0.8x | 1.1x → 弹回 | 3帧（恢复快） |

### Special Animations
- **半鬼化（>80怒）**: 角持续发红光，眼睛变橙色，皮肤紫色加深10%，呼吸时口鼻冒紫色气
- **完全鬼化（必杀后5秒）**: 身体放大15%，角全部亮起，眼睛血红，暗紫色气焰环绕全身，每步脚印留下裂纹
- **必杀·鬼王降临**: 跳起到屏幕外→巨锤砸下→地面塌陷→连砸7次（每次裂缝扩大）→最终一击时屏幕全暗→只有锤子和角在发光→最后一击全屏白→鬼族虚影浮现

### Topics小鬼丸 Can Discuss
- **打架**: "你好强！跟我打一架！打完我们就做朋友！"
- **食物**: "这个好吃！你也吃！（递过去一块不明物体）"

## Lore Connections

- **熔岩地狱**: 小鬼丸的出生地，一个充满鬼族的地下世界
- **逃跑原因**: "太热了！不想待了！" （实际上是因为鬼族内部的某个事件，但小鬼丸不愿提起）
- **锤子"小糖"**: 从地狱带出来的唯一物品，据说是用"永冻糖晶"锻造的——地狱中最坚硬的材料
- **与其他角色**:
  - 对武僧: "你打架好快！教我教我！" （但学不会）
  - 对魔导师: "你的糖好漂亮！能吃吗？" "不能吃！" "哦……"
