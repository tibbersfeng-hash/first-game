# Character: 糖果魔导师 · 糖糖

## Quick Reference

- **Full Name**: 糖糖（Tangtang）
- **职业代号**: Sugar Hexblade
- **Role in Story**: 主角备选 — 天才糖果炼金术士，用甜食征服世界（字面意思）
- **Role in Gameplay**: 可玩主角职业 #2 — 中程控场型
- **First Appearance**: 副本选择画面（可选角色）
- **Status**: Provisional

---

## Concept

糖糖是一个看起来甜美可人的小魔女，但她调制的糖果可不是用来吃的——每一颗都是高浓缩的魔法炸药。她用一根巨型棒棒糖法杖在安全距离外倾泻糖果弹幕，用地雷封锁走位，用甜蜜的外表诱骗敌人走进死亡陷阱。

玩家应该喜欢糖糖的原因：**甜蜜陷阱的快感**——看着敌人在自己布置的糖果地雷阵中手足无措，然后一颗魔法糖弹精准引爆，满屏糖果色粒子爆炸，又美又致命。

## Personality

### Core Traits
- **腹黑甜美** — 永远带着天使般的微笑，然后毫不留情地炸飞敌人
- **完美主义** — 对糖果炼金术的要求极高，"温度差一度都不行！"
- **收藏癖** — 看到稀有材料就走不动路，"这个可以做新的糖果配方！"
- **毒舌** — 用最甜的语气说最狠的话

### Voice Profile
- **Speech Pattern**: 甜腻礼貌，句尾带"~"，但内容经常很暴力
- **Vocabulary Level**: 混合炼金术语和甜食词汇
- **Verbal Tics**: "嗯哼~" / "炸飞你哦~" / "这个配方还差一味~"
- **Tone Reference**:  cute外表 + 冷静计算 + 偶尔黑化

### Emotional Range

| Emotion | Trigger | Expression | Example Line |
|---------|---------|-----------|-------------|
| 开心 | 发现新材料 | 眼睛变成心形，法杖冒泡泡 | "嗯哼~好东西~让我试试~" |
| 生气 | 糖果被弄坏 | 微笑不变但眼睛失去高光 | "你……弄坏了我的糖果呢~要付出代价哦~" |
| 得意 | 连环爆炸 | 双手叉腰，帽子铃铛响 | "完美配方！完美爆炸！嗯哼~" |
| 困惑 | 敌人没踩地雷 | 歪头，法杖戳地 | "奇怪……不应该啊……走那边才对呀~" |
| 害怕 | Boss太大只 | 躲到法杖后面，只露出眼睛 | "好、好大……不过大的目标更容易命中呢~" |

## Motivation and Arc

### Primary Motivation
追求"终极糖果配方"——传说地下城的深处藏着远古炼金术士留下的"永恒之糖"，那是世界上最强大的魔法糖果。

### Character Arc (Lightweight)

| Phase | State | Trigger |
|-------|-------|---------|
| 初始 | 旅行炼金术士，寻找稀有材料 | 角色选择 |
| 成长 | 发现地下城中的远古配方线索 | 通关幽灵古堡 |
| 蜕变 | 面对"终极配方"的代价 | 通关熔岩地狱 |
| 终局 | 得到永恒之糖，成为传奇炼金术士 | 通关最终副本 |

### Internal Conflict
糖糖追求最强糖果，但她内心知道——终极配方的力量可能失控。"我是为了做最好吃的糖，还是最危险的糖？" 这个矛盾在剧情中以轻松的方式呈现。

## Gameplay Function

### 职业定位
**中程控场型** — 保持安全距离，用糖果弹幕和地雷控制战场节奏。

### 核心机制：糖果配方

```
糖果配方（Candy Combo）：3种糖果类型切换
- 红糖弹（直射弹丸，中速，中伤害）
- 蓝糖雷（放置型，踩中后延迟爆炸，高伤害）
- 绿糖胶（减速区域，低伤害，控制走位）
- 切换方式：闪避时按攻击键 = 切换糖果类型
- 连招中混合使用不同糖果可触发"配方奖励"（额外伤害+20%）
```

### 攻击风格

| 攻击 | 动作描述 | 特效 | 手感 |
|------|----------|------|------|
| 轻攻1 | 法杖挥出单颗红糖弹 | 粉色弹丸+尾迹 | 轻快弹射 |
| 轻攻2 | 法杖旋转，扇形散射3颗 | 三色弹丸扇形扩散 | 范围覆盖 |
| 轻攻3 | 巨大糖果炮弹（蓄力1帧） | 大型糖果炮弹+爆炸 | 中等冲击 |
| 重攻 | 放置蓝糖地雷（脚下） | 蓝色发光地雷+脉冲 | 策略部署 |
| 必杀 | 彩虹糖果大风暴 | 全屏糖果弹幕雨+彩虹光束 | 华丽毁灭 |
| 闪避 | 糖雾瞬移（留下一团糖果雾） | 粉色烟雾+瞬移残影 | 诡秘灵动 |

### 连招路径

```
基础连招（安全输出）:
J → J → J（红糖弹连射）→ 等敌人靠近 → 踩雷（重攻提前放置）

进阶连招（配方混合）:
J（红糖弹）→ 放置地雷 → J-J（逼敌人走向地雷）→ 引爆

高手连招（全类型混合）:
放置3个地雷 → 绿糖胶封路 → 红糖弹驱赶 → 敌人踩雷连环爆炸 → L（必杀收割）
```

### What This Character Provides
- **最安全的输出** — 攻击距离全职业最远，可以在屏幕边缘输出
- **最强控场** — 地雷和糖胶可以封锁Boss移动路线
- **策略深度** — 不是反应速度，而是布局思维
- **风险/回报高** — 被近身很脆弱，但控场好几乎无伤

## Animation Feel

### Idle（8帧循环）
1. 歪头舔棒棒糖（法杖）
2-3. 吐出一颗小糖果弹→小爆炸→满意点头
4-5. 法杖在空中画圈，留下彩虹轨迹
6-7. 从腰间瓶子倒出糖果检查颜色
8. 举起法杖，帽铃叮当

### Attack Deformation Rules
| 动作 | 横向缩放 | 纵向缩放 | 持续时间 |
|------|----------|----------|----------|
| 轻攻挥杖 | 1.1x | 0.95x | 3帧 |
| 重攻放雷 | 0.9x | 1.1x | 5帧 |
| 必杀蓄力 | 0.7x | 1.3x | 8帧 |
| 必杀释放 | 法杖伸长1.5x | 屏幕上方倾泻 | 持续 |
| 受击 | 0.7x | 1.0x → 弹回 | 4帧 |

### Special Animations
- **放置地雷**: 蹲下→法杖点地→蓝色魔法阵→糖果地雷浮起→脉冲发光
- **配方切换**: 法杖顶端糖果旋转变色（红→蓝→绿循环）
- **必杀**: 跳起→法杖指向天空→彩虹色魔法阵展开→糖果如雨倾泻→最后一道光柱贯穿
