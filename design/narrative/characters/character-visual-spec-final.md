# 角色视觉定稿：格斗萌主

> **Status**: FINAL — 视觉定稿 v2.0
> **Author**: narrative-director + art-director
> **Last Updated**: 2026-06-18
> **Version**: 2.0（正常比例版，v1.0 为 Q版）
> **用途**: 3D 美术制作的唯一视觉参考标准，所有角色模型/动画/特效必须遵循本文档

---

## 全局规范

### 比例
- **头身比**: 7-8 头身（标准人形比例）
- **慧空**: 7.5 头身，身材修长偏少年感
- **糖糖**: 6.5 头身，娇小萝莉体型
- **鬼斩**: 7 头身，矮壮结实
- **面部**: 日式动漫风格，大眼但不夸张，面部有立体感
- **体型**: 保留角色个性特征，不做完全写实

### 美术风格
- **风格**: 日式动漫风格化（Cel-shaded NPR）
- **着色**: 3D 卡通渲染，无描边或极细描边
- **色彩**: 高饱和糖果色技能特效 + 暗色地牢场景对比
- **参考**: 崩坏3（主）、原神（辅）、绝区零

### 表情系统（通用）
| 表情 | 眼睛 | 嘴巴 | 使用场景 |
|------|------|------|----------|
| 正常 | 动漫大眼+高光 | 微笑 | idle, 走路 |
| 战斗 | 眼神锐利，眉毛微压 | 抿嘴/张口 | 攻击 |
| 愤怒 | 眉头紧锁，瞳孔收缩 | 咬牙 | 被击，蓄力 |
| 得意 | 眯眼弯月 | 大笑 | 必杀收招，通关 |

## 角色 1: 气功少年 · 慧空

### 剪影描述
一个清秀的少年武僧，光头，头顶隐约有金色气功纹路。红色头带从脑后飘出2条长飘带。脖子上挂着一串念珠（每颗直径约5cm）。穿深蓝色交领武僧服，红色宽腰带。双拳缠着金色绑带。身材修长，腿长，动作敏捷。

| 内衬 | 白色 | #F5F5F5 | V领内衬 |
| 腰带 | 鲜红 | #CC3333 | 宽腰带 |
| 头带 | 鲜红 | #DD3333 | 包括飘带 |
| 念珠 | 深棕 | #8B6914 | 带金色光泽 |
| 念珠光效 | 金色 | #FFD700 | 施法时发光 |
| 护腕 | 深灰 | #444444 | 包裹前臂末端 |

### 体型特征
- **身高**: 约 170cm（7.5头身）
- **体型**: 偏瘦但有肌肉线条（武僧练功成果）
- **肩宽**: 适中，不过宽
- **腿长**: 修长，占身高 50%+
- **发型**: 完全光头，无头发
- **面部**: 清秀少年，剑眉星目

## 角色 2: 糖果魔导师 · 糖糖

### 剪影描述
一个娇小的少女魔导师，最突出的特征是头上那顶比头还大的尖顶魔法帽（帽尖弯曲，挂着小铃铛）。粉色双马尾从帽子两侧垂下。手持一根巨型棒棒糖法杖（棒棒糖部分比她的头还大，彩虹螺旋纹理）。穿粉色魔法袍+白色围裙。身材娇小但比例正常。

## 角色 3: 萌鬼狂战士 · 鬼斩

### 剪影描述
一个矮壮的少年鬼族，头顶两只短短的小鬼角（像小珊瑚）。体型比其他两个角色宽20%，看起来结实圆润。穿红色肚兜+铁肩甲（左大右小不对称），扛着一把比自己大三倍的巨型糖果锤。锯齿状嘴巴，微笑时露出小尖牙。

### 色彩规范

| 部位 | 颜色名 | Hex | 备注 |
|------|--------|-----|------|
| 皮肤 | 浅紫 | #B39DDB | 全身皮肤（鬼族特征） |
| 皮肤暗面 | 深紫 | #7E57C2 | 背光面 |
| 鬼化皮肤 | 深紫+ | #5E35B1 | 怒气>80时 |
| 肚兜 | 暗红 | #C62828 | 胸前有黑色"鬼"字纹 |
| 左肩甲 | 深灰铁 | #424242 | 大肩甲（不对称） |
| 右肩甲 | 深灰铁 | #424242 | 小肩甲 |
| 护臂 | 黑铁 | #212121 | 前臂+手腕 |
| 锤子糖果头 | 彩虹条纹 | 多色交替 | 糖果造型但重金属质感 |
| 锤子柄 | 深棕 | #5D4037 | 粗木柄 |
| 短裤 | 深灰 | #424242 | 宽松短裤 |
| 腰带绳 | 粗麻色 | #A1887F | 粗绳束腰 |
| 铃铛 | 金色 | #FFD700 | 脖子上小铃铛（反差萌） |
| 角（常态） | 鲜红 | #FF1744 | 两只短角 |
| 角（蓄怒） | 金红发光 | #FF6D00 | 怒气>80 |
| 眼睛（常态） | 金色 | #FFD600 | 圆眼 |
| 眼睛（鬼化） | 血红 | #FF0000 | 怒气=100 |

### 体型特征

- **体型**: 矮壮结实，肩宽腰粗
- **肩宽**: 比其他角色宽20%
- **腿长**: 正常比例，腿较短显壮
- **发型**: 短发（深色）
- **面部**: 锯齿嘴，小尖牙，憨厚但凶

## 3D 资产规范

### 模型规范
| 参数 | 值 |
|------|-----|
| 面数 | 30,000 - 100,000 三角面 |
| 贴图分辨率 | 2048×2048 (PBR) |
| 骨骼数 | < 75 根 |
| 格式 | FBX (UE5 导入) |
| 绑定 | Mixamo / 手工绑骨 |

| 参数 | 值 |
|------|-----|
| 帧率 | 30 fps |
| 动画格式 | FBX / UE5 AnimSequence |
| 根运动 | 启用 (Root Motion) |

| 参数 | 值 |
|------|-----|
