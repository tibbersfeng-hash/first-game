# Character: 气功小武僧 · 慧空

## Quick Reference

- **Full Name**: 慧空（Huikong）
- **职业代号**: Blazing Monkfist
- **Role in Story**: 主角 — 初入江湖的武僧弟子，踏上成为格斗大师之路
- **Role in Gameplay**: 可玩主角职业 #1 — 近战连招型，MVP首发职业
- **First Appearance**: 教程副本「觉醒之拳」
- **Status**: Provisional

---

## Concept

慧空是一个在深山寺庙长大的小武僧，天赋异禀却生性贪玩。他偷偷溜出寺庙闯荡江湖，用一双小拳头和一颗赤子之心在地下城中打出一片天。

玩家应该喜欢慧空的原因：**反差萌的极致**——看起来是个圆滚滚的可爱小光头，出拳时却是金刚降世般的爆裂打击。每一个连招都是一场视觉烟花秀。

## Appearance

### Build
- 三头身Q版比例，身高约角色标准单位（头:身:腿 = 1:1:1）
- 体型偏瘦但肌肉线条隐约可见（小和尚练功的成果）
- 光头顶，无头发，但有金色气功纹路在施放技能时浮现

### Distinguishing Features
- **光头 + 金色气功纹** — 最突出的辨识度，施放必杀时头顶纹路发出金光
- **超大念珠手串** — 挂在脖子上，珠子比拳头还大，idle 时晃动
- **红色头带 + 飘带** — 从头部后方飘出，战斗中飘动增加动感
- **拳套** — 缠着金色绑带的拳头，蓄力时发光

## Personality

### Core Traits
- **天真热血** — 对一切都充满好奇和热情，看到强大的对手就想挑战
- **贪吃好动** — 无法安静超过30秒，总在找吃的或找打的
- **正义感强** — 看到不公平的事就忍不住出手，虽然有时候帮倒忙
- **武痴** — 对武术有天然的理解力，但理论课从来不及格

### Voice Profile
- **Speech Pattern**: 短句、感叹句为主，语气上扬，充满活力
- **Vocabulary Level**: 简单直白，偶尔蹦出佛经词汇（但用错语境）
- **Verbal Tics**: "阿弥陀佛，吃我一拳！" / "师傅说这个不能打……那我轻轻打？"
- **Tone Reference**: 路飞的乐观 + 悟空的好战 + 小和尚的天真

### Emotional Range

| Emotion | Trigger | Expression | Example Line |
|---------|---------|-----------|-------------|
| 兴奋 | 遇到强敌 | 眼睛变成星星状，拳头冒金光 | "哇！你好强！再来再来！" |
| 愤怒 | 看到欺负弱小 | 气功纹全亮，表情变严肃 | "不许欺负人！金刚怒目！" |
| 沮丧 | 连招失败 | 耳朵（？）下垂，念珠不晃了 | "啊……师傅说得练更久……" |
| 得意 | 打出S评分 | 双手叉腰，闭眼微笑 | "嘿嘿，我也不赖嘛！" |
| 饥饿 | 进副本前 | 摸肚子，念珠变成食物形状 | "打完这关去吃包子好不好？" |

## Motivation and Arc

### Primary Motivation
证明自己的实力——不是给师傅看，而是给自己看。慧空想知道：一个贪吃的小和尚，能不能成为传说中的格斗大师？

### Character Arc (Lightweight)

| Phase | State | Trigger |
|-------|-------|---------|
| 初始 | 天真的武僧弟子，偷偷下山 | 游戏开始 |
| 成长 | 在副本中认识到实力差距 | 第一次被Boss打败 |
| 蜕变 | 将寺庙修行与实战融合 | 通关熔岩地狱 |
| 终局 | 成为新的格斗传说 | 通关最终副本 |

### Internal Conflict
寺庙教导"慈悲为怀"，但战斗需要"全力以赴"。慧空的成长之路就是在慈悲与力量之间找到自己的平衡。

## Gameplay Function

### 职业定位
**近战连招型** — 近距离快速输出，以连击数和取消链的深度为核心。

### 核心机制：气功层数

```
气功层数（Qi Stacks）：0-5层
- 每次命中+1层（轻攻+1，重攻+2）
- 每层提供+5%攻击速度（最多+25%）
- 层数在5秒未命中后开始衰减（每秒-1层）
- 5层时轻攻第三段自动变为"气功爆发拳"（伤害×2，带击退）
```

### 攻击风格

| 攻击 | 动作描述 | 特效 | 手感 |
|------|----------|------|------|
| 轻攻1 | 快速直拳（右） | 白色小型气功弧 | 快如闪电 |
| 轻攻2 | 勾拳（左） | 蓝色弧形气劲 | 略微上挑 |
| 轻攻3 | 升龙拳（双拳上勾） | 金色龙形气功波 | 击飞效果 |
| 重攻 | 气功掌（前推双掌） | 金色圆形冲击波 | 沉重推击 |
| 必杀 | 如来神掌·金佛降临 | 全身金光→巨型金色佛掌从天而降 | 全屏打击感 |
| 闪避 | 缩身滚动（金色残影） | 金色半透明残影 | 灵巧翻滚 |

### 连招路径

```
基础连招（入门）:
J → J → J → K（直拳→勾拳→升龙→气功掌）
    ↓
进阶连招（取消链）:
J → J → [cancel] K → [cancel] L（重攻取消→必杀）
    ↓
高手连招（气功5层）:
J → J → J(气功爆发拳) → [cancel] K → [cancel] 闪避 → J → K → L
```

### What This Character Provides
- **最快的手感** — 轻攻startup仅2帧，全职业最快
- **最深的取消链** — 因为攻速快，取消窗口更多，操作空间最大
- **连击数之王** — 同时间内打出的连击数最多，评分最高潜力
- **风险/回报中等** — 需要近身，但闪避灵活

## Animation Feel

### Idle（8帧循环）
1. 双手合十闭眼（禅定状）
2-3. 突然睁眼，摆出战斗架势
4-5. 左右跳步，拳套发出微光
6-7. 念珠晃动，甩甩拳头
8. 回到战斗站姿

### Attack Deformation Rules（打击变形）
| 动作 | 横向缩放 | 纵向缩放 | 持续时间 |
|------|----------|----------|----------|
| 轻攻出拳 | 1.2x | 0.9x | 3帧 |
| 重攻推掌 | 1.3x | 0.7x | 4帧 |
| 必杀蓄力 | 0.7x | 1.3x | 8帧 |
| 必杀释放 | 1.5x | 0.6x → 弹回 | 6帧 |
| 受击 | 0.6x | 1.1x → 弹回 | 4帧 |

### Special Animations
- **5层气功满层**: 头顶金色纹路脉动发光，念珠悬浮旋转
- **必杀蓄力**: 双掌合拢→金光从指缝溢出→地面碎石浮起
- **必杀释放**: 巨型金色佛掌从天而降（占屏幕1/3），砸地时全屏闪白

## Lore Connections

- **师傅**: 深山古寺的老武僧，传说是上一代格斗大师，但因某种原因隐居
- **寺庙**: 慧空长大的地方，是江湖中"气功一脉"的传承地
- **念珠**: 师傅给的遗物（？），据说有特殊的能量，但慧空只知道它能砸人
- **格斗大师传说**: 江湖传说中，集齐五大职业的力量可以打开"武道之门"
