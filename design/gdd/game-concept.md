*Created: 2026-06-15*
*Last Updated: 2026-06-17 (3D 转型)*

## Elevator Pitch

> 动漫风格3D第三人称动作闯关游戏，玩家操控萌系角色在副本中搓招连击，通过职业成长和装备搭配从萌新冒险者成长为传说中的格斗大师，还能随时和好友来一场1v1切磋。

## Core Fantasy

成为一名动漫风格格斗大师——在3D副本空间中闪转腾挪、搓出华丽连招，从无名小卒一路打到传奇段位。每次副本都是一场爽快的演出，每次切磋都是操作与build的交锋。糖果色特效与暗色地牢形成鲜明对比，每次连招都是一场视觉盛宴。

---

## Unique Hook

崩坏3式的3D动作体验，AND ALSO 糖果炸裂风让视觉更独特——动漫风格角色在暗色地牢中释放高饱和糖果色技能，必杀释放时镜头拉近、特写演出、糖果色爆炸填满屏幕。相比崩坏3的复杂操作，我们简化搓招门槛，3-4键组合就能打出爽快连击，同时保留3D空间闪避、锁定切换、空中连招等深度。

## Player Experience Analysis (MDA Framework)

### Target Aesthetics (What the player FEELS)

| Aesthetic | Priority | How We Deliver It |
| ---- | ---- | ---- |
| **Sensation** | 1 (核心) | 3D夸张打击感、镜头震动+拉近特写、伤害数字弹射、动漫表情变形、糖果色爆裂特效 |
| **Challenge** | 2 (核心) | 连招精通、3D空间走位、Boss机制、PVP段位 |
| **Fantasy** | 3 (核心) | 动漫风格格斗大师的角色扮演、职业身份认同 |
| **Expression** | 4 (支撑) | 装备build搭配、技能树分支、外观自定义 |
| **Fellowship** | 5 (支撑) | 好友1v1切磋、排行竞争 |
| **Discovery** | 6 (次要) | 3D副本探索、隐藏区域、装备词缀组合 |
| **Narrative** | 7 (次要) | 轻量剧情，服务于世界观氛围 |
| **Submission** | N/A | 不追求放松体验 |

### Key Dynamics (Emergent player behaviors)

- 玩家会反复练习连招组合以追求更高评分和更爽的打击体验
- 玩家利用3D空间走位闪避Boss攻击，绕后输出
- 玩家会在PVE和PVP之间切换：PVE练build，PVP验操作
- 玩家会尝试不同职业体验截然不同的战斗风格
- 玩家会在装备词缀组合中寻找冷门强力build

### Core Mechanics (Systems we build)

1. **3D连招战斗系统** — 3D空间闪避+锁定切换+简化搓招（3-4键），命中反馈层（停顿+镜头拉近+数字+特效），评分系统
2. **副本闯关系统** — 3D竞技场式房间推进，Boss机制，掉落与评分
3. **职业与成长系统** — 多职业，技能树分支，装备与词缀
4. **轻量PVP系统** — 1v1竞技场，段位排名，好友切磋
5. **城镇与社交系统** — 副本间休整地，排行榜，好友列表

## Player Motivation Profile

### Primary Psychological Needs Served

| Need | How This Game Satisfies It | Strength |
| ---- | ---- | ---- |
| **Competence** | 连招精通、副本S评、PVP段位，清晰的技能成长曲线 | Core |
| **Autonomy** | 职业选择、技能树分支、装备build自由搭配 | Core |
| **Relatedness** | 好友切磋、排行竞争、分享build | Supporting |

### Flow State Design

- **Onboarding curve**: 第一个教程副本教普攻+1个技能+3D闪避，3分钟内打出第一个连招
- **Difficulty scaling**: 副本分普通/困难/地狱，Boss机制逐层加复杂度
- **Feedback clarity**: 连招数实时显示、评分即时反馈、伤害数字颜色分级
- **Recovery from failure**: 复活币机制，失败后3秒重进，不惩罚探索

## Core Loop

### Moment-to-Moment (30 seconds)
3D空间走位 → 锁定目标 → 按键输入连招 → 角色释放技能/普攻 → 命中判定 → 打击反馈（停顿+镜头拉近+震动+数字+表情特写） → 连招计数器+1 → 切换目标/评估下一步

### Short-Term (5-15 minutes)
进入副本区域 → 清怪（追求S评连招） → 掉落拾取 → 推进到下一区域 → 遇到精英/Boss → 3D空间机制应对 → 房间结算评分

### Session-Level (30-120 minutes)
城镇整备（装备/技能/商店） → 选择副本 → 完整通关 → 结算奖励 → 回城镇 → 选择：再来一次/换副本/PVP切磋/下线

### Long-Term Progression
- **力量成长**: 装备强化、属性提升、新技能解锁
- **知识成长**: 掌握职业连招套路、Boss机制记忆、PVP对策
- **选择成长**: 技能树分支选择、装备词缀build搭配
- **社交成长**: PVP段位提升、排行榜排名

## Game Pillars

### Pillar 1: 爽快打击 + 糖果炸裂
每次连招都是微型演出。3D空间让打击感更立体——镜头拉近特写、角色被冲击波震退变形、糖果色粒子爆炸填满整个屏幕。暗色地牢与高饱和技能形成强烈视觉冲击。

*Design test*: 如果要选"加一个新职业"还是"优化打击感反馈"，选打击感。

### Pillar 2: 职业深度与成长
多职业多build，装备搭配驱动长期追求，每个职业玩法截然不同。3D空间为每个职业提供独特的空间玩法（武僧贴脸连击、魔导师远程控场、狂战士AOE震地）。

*Design test*: 如果要选"加新副本"还是"加新技能树分支"，选技能树。

### Pillar 3: 轻社交竞争
好友切磋、排行竞争，但不强制组队，社交是调味料不是主菜。

*Design test*: 如果要选"做多人团本"还是"做1v1竞技场"，选1v1。

### Anti-Pillars (What This Game Is NOT)

- **NOT 重度社交**: 不做大型团本、公会战、强制组队。这会妥协"爽快打击"支柱——团本中个人操作感被稀释。
- **NOT 硬核操作门槛**: 连招简化到3-4键组合，新手也能爽。追求崩坏3高级分支操作深度会吓走轻度受众。
- **NOT 开放世界**: 副本制闯关，不做大地图探索。开放世界会稀释核心战斗密度。
- **NOT 纯2D横版**: 完全3D空间体验，不是2.5D伪3D。

## 3D 设计原则

基于崩坏3等参考总结的核心3D设计原则：

### Camera (镜头)
- **默认视角**: 第三人称越肩（over-shoulder），距离角色约2-3米
- **战斗视角**: 锁定敌人时镜头自动拉近，敌人位于屏幕中心偏上
- **闪避镜头**: 极限闪避时触发子弹时间+镜头轻微拉近（崩坏3式）
- **必杀演出**: 触发必杀时镜头拉近至角色特写，背景虚化
- **自由视角**: 右摇杆/鼠标可自由旋转（非战斗时）

### Movement (移动)
- **基础移动**: WASD/左摇杆全向移动，带加速/减速曲线
- **闪避**: 空格键/手柄B，3D空间全向闪避（无敌帧）
- **跳跃**: 单段跳，空中可攻击
- **冲刺**: 双击方向键或Shift，快速位移（消耗体力）
- **锁定移动**: 锁定敌人时，移动方向相对敌人（环绕走位）

### Combat Space (战斗空间)
- **竞技场式**: 每个战斗房间是有限3D空间（类似崩坏3的副本房间）
- **边界**: 隐形墙限制活动范围，保证战斗密度
- **垂直空间**: 平台、高低差增加空间维度（但不是核心）
- **环境互动**: 可破坏物件、陷阱、地形优势

### Target Lock (锁定系统)
- **自动锁定**: 攻击时自动锁定最近敌人
- **手动切换**: 按Tab/手柄R切换锁定目标
- **锁定UI**: 被锁定敌人头上显示标记
- **无锁定模式**: 按住锁定键时锁定，松开恢复自由（可选）

## Risks and Open Questions

### Design Risks
- 连招简化后深度不足，老玩家觉得"太简单"不够爽
- 3D空间的战斗可能失去2D横版的"精准感"
- PVP平衡难度大，3D空间+不同职业+build之间的公平性难以保证
- 副本重复感——竞技场式闯关如果缺乏变化容易腻

### Technical Risks
- 3D打击手感调优比2D更复杂（动画+碰撞+镜头协调）
- 镜头系统在复杂场景中可能穿模/抖动
- PVP网络延迟对3D动作游戏同样致命，同步方案需要早期验证
- 3D美术资产产出速度慢于2D sprite

### Market Risks
- 崩坏3/原神已是巨兽，同品类竞争激烈
- Q版可能被误判为"儿童向"，需要营销上明确动作硬核定位（现已改为正常比例，降低此风险）

### Scope Risks
- 3D美术资产工作量巨大——每个职业需要完整3D模型+动画集
- 多职业设计范围容易膨胀——每个职业都意味着一套完整3D资产+技能+平衡
- 装备系统如果做词缀随机，内容量级指数增长

### Open Questions
- 3D连招系统到底简化到什么程度？需要原型验证
- 锁定系统采用"按住锁定"还是"切换锁定"？需要手感测试
- PVP用帧同步还是状态同步？需要技术调研
- 3D角色是用Metahuman还是手建模型？影响美术管线效率

## MVP Definition

**Core hypothesis**: 玩家觉得动漫风格3D角色的连招战斗本身足够爽，愿意为了更高评分反复刷副本。

**Required for MVP**:
1. 1个职业的完整3D连招战斗系统（普攻+轻/重+3个技能+必杀+闪避）
2. 第三人称镜头系统（自由视角+战斗锁定）
3. 1个3D竞技场式副本（含1个Boss）
4. 评分系统（连招数+通关时间+无伤加分）
5. 基础打击反馈（命中停顿+镜头拉近+屏幕震动+伤害数字）

**Explicitly NOT in MVP**:
- 多职业（只做1个验证核心）
- 装备系统
- PVP
- 城镇/商店

### Scope Tiers (if budget/time shrinks)

| Tier | Content | Features | Timeline |
| ---- | ---- | ---- | ---- |
| **MVP** | 1职业1副本 | 3D连招战斗+镜头+评分+打击反馈 | 6-8周 |
| **Vertical Slice** | 2职业3副本 | MVP+装备系统+城镇 | 4-5月 |
| **Alpha** | 3职业10副本 | 全功能粗版 | 8-12月 |
| **Full Vision** | 5+职业20+副本 | 全功能打磨+PVP | 14-20月 |

## Visual Identity Anchor

**方向**: 糖果炸裂风 — 动漫风格3D角色在暗色地牢中释放糖果色爆裂技能

**视觉规则**: 一切必须"萌系外壳+硬核内芯"——角色设计偏萌系可爱，但技能特效和打击反馈是尖锐爆裂的

**支撑原则**:
1. **软硬反差** — 角色设计偏萌，特效锋利炸裂（设计测试：新特效如果只是"好看"但不"炸"，则不符合）
2. **暗底亮效** — 地牢暗色调，技能和伤害数字是高饱和亮色（设计测试：如果关掉特效画面变暗沉，则正确）
3. **3D夸张表现** — 命中时角色瞬间受力反馈（崩3式形变），打击感的关键
4. **镜头演出** — 必杀/重击时镜头拉近特写，背景虚化，角色表情清晰可见
