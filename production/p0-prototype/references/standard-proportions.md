## 各职业差异

### 气功小武僧 (Huikong) — 男性 7.5 头身

```
特征: 修长有力, 光头, 飘带
参考: 崩坏3 琪亚娜/八重樱 体型 + 武僧元素
- 头: 光头或极短发, 额间红点, 眉毛浓密有型
- 躯干: 修长但肌肉线条明显 (胸肌/腹肌轮廓)
- 手臂: 修长有力, 前臂绑带, 戴气功拳套
- 腿: 修长, 肌肉线条流畅
- 飘带: 从腰部飘出, 长而飘逸
- 整体: 修长匀称, 动作流畅, 参考 李小龙/叶问 气质
```

### 糖果魔导师 (Tangtang) — 女性 7.0 头身

```
特征: 纤细灵动, 尖帽, 法杖
参考: 原神 凝光/刻晴 体型
- 头: 长发或双马尾, 戴小尖帽
- 躯干: 纤细, 腰细, 胸线柔和
- 手臂: 纤细, 袖口宽大飘逸
- 腿: 修长, 穿短裙 + 长靴
- 法杖: 糖果造型, 比人高
- 整体: 修长优雅, 甜美灵动
```

### 萌鬼狂战士 (Guizhan) — 男性 7.5 头身 (壮硕版)

```
特征: 壮硕威猛, 鬼角, 巨锤
参考: 崩坏3 德丽莎/原神 荒泷一斗 壮硕版
- 头: 有鬼角 (额前两侧), 短发或光头
- 躯干: 最壮 (比标准宽 15-20%), 肌肉明显
- 手臂: 粗壮, 戴重护腕
- 腿: 粗壮, 重靴
- 巨锤:  oversized 武器, 比人高大
- 整体: 宽厚沉稳, 压迫感强
```

## 面部特征 (动漫风格化)

```
        眉毛 (修长有型)
      ──────────────
    ┌────────────────────┐
    │  ◉              ◉  │  大眼 (占脸宽 1/4)
    │  ●               ● │  瞳孔
    └────────────────────┘
           \    /
            ‾‾‾‾          小巧鼻梁 (简化)
             ‿  ‿         嘴 (正常大小)
```

**关键比例**:
- 眼睛: 占脸宽 1/4（比写实时大，比 Q 版小）
- 鼻子: 简化为线条或小巧形状
- 嘴巴: 正常大小，位于脸部下方 1/3
- 眉毛: 修长有型，表情丰富

## NPR 渲染注意事项

- 描边粗细: 正常比例描边比 Q 版细，更精致
- 阴影分层: 2-tone 或 3-tone，边界柔和
- Rim Light: 发丝和衣服边缘重点表现
- 材质: PBR base + NPR shading 混合

## 建模检查清单

- [ ] 总高 175-185cm (7-7.5 头高)
- [ ] 头高 25cm (比例基准)
- [ ] 腿部长度 ≥ 全身 1/2 (修长感)
- [ ] 肩宽与头宽比例正确
- [ ] 动漫风面部比例 (大眼、小鼻、正常嘴)
- [ ] 职业特征明显 (武僧飘带/魔导师帽子/狂战士鬼角)
- [ ] T-Pose 正确 (用于绑骨)
- [ ] 面数控制: 角色 < 50k 三角面
