# 气功小武僧 (Huikong) 设计参考

> **用途**: P0 原型建模参考
> **职业**: 气功小武僧 / Blazing Monkfist
> **定位**: 近战连招型
> **比例**: 7.5 头身（正常偏修长）
> **风格**: 动漫风格化 正常比例（参考 崩坏3 / 原神 / 绝区零）

## 角色概念

**背景**: 来自糖门寺的年轻武僧，精通气功与近身格斗。
外表修长俊朗，内心平静坚定。招式刚柔并济，以气功波和连击见长。

**性格关键词**: 沉静、敏捷、刚柔并济

**参考角色**:
- 崩坏3: 琪亚娜/芽衣（体型）+ 符华（武僧气质）
- 原神: 钟离（修长体型）+ 重云（气质）
- 经典: 李小龙、叶问、少林武僧

### 细节分解

| 部位 | 设计元素 |
|---|---|
| **头部** | 光头或极短寸头，额间红点（戒疤变体，标志性特征），耳垂不拉长（写实化） |
| **面部** | 浓密上扬眉毛，琥珀色/深棕色眼睛，平静有神，轮廓分明 |
| **颈部** | 修长，无念珠（简化） |
| **上装** | 深蓝僧衣，无袖或短袖（露出手臂），斜襟交领，金橙镶边 |
| **手腕** | 金色护腕，前臂绑带缠绕 |
| **手部** | 露指气功手套（露出指尖，用于释放气功波） |
| **腰部** | 金橙宽腰带，武术结法，飘带自然下垂 |
| **下装** | 白色宽松武术裤，裤脚收紧束进靴子 |
| **脚** | 深棕长靴（到小腿中部），武术靴薄底灵活 |
| **飘带** | 2 条橙金色飘带，从腰带后方飘出，长及膝，末端渐变分叉 |

## 动画风格

### 待机 (Idle)
```
- 平静站立，轻微呼吸起伏
- 偶尔活动手腕或闭目养神
- 飘带缓慢飘动
- 重心微微移动（武术架势感）
```

### 行走 (Walk)
```
- 双手握拳或自然武术架势
- 步伐灵活（武术步法）
- 飘带随走动飘动
- 身体稳定，无上下弹跳（沉稳感）
```

### 轻攻 (Light Attack)
```
Light 1: 左直拳 (快速, 前冲, 带残影)
Light 2: 右勾拳 (接上, 身体旋转, 气流痕迹)
Light 3: 双掌推 (气功波起手, 金色光环扩散)

每招: 蓄力 0.1s → 出击 0.1s → 收招 0.3s
参考: 李小龙/叶问 出拳节奏
```

### 重攻 (Heavy Attack)
```
气功波:
- 双手胸前合十 0.5s
- 蓝色/金色气功能量聚集 0.5s
- 推出, 冲击波 0.5s
- 收招 0.5s
```

### 必杀 (Ultimate)
```
如来神掌:
- 跳起, 镜头拉近 1s
- 空中双手合十, 金光环绕 1s
- 俯冲, 巨掌从天而降 1s
- 爆炸特效, 镜头震动 1s
```

### 闪避 (Dodge)
```
气功瞬移:
- 身体化为蓝光 0.1s
- 瞬移到目标方向 0.1s
- 重新凝结 0.1s
- 残留蓝色残影
```

## 建模优先级

P0 阶段只建模核心可见元素:

1. **基础身体** (头+躯干+手臂+腿) — Day 1-3（可用 tccli ai3d）
2. **僧衣基础** (简单衣服模型) — Day 3-4
3. **关键配件** (护腕、飘带、手套) — Day 4-5
4. **武器** (气功拳套 — 简单即可) — Day 5-6
