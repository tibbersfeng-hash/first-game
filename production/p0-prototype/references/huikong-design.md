# 气功小武僧 (Huikong) 设计参考

> **用途**: P0 原型建模参考
> **职业**: 气功小武僧 / Blazing Monkfist
> **定位**: 近战连招型

## 视觉特征

```
主色: 蓝 #6699FF
辅色: 金/白
特效色: 金色气功波
糖果色: 蓝色冰糖
```

### 整体轮廓

```
    ╱╲
   ╱  ╲     ← 光头(圆) + 头饰
  ┌────┐
  │    │     ← 脸(圆)
  ╰────╯
   ╱  ╲     
  ╱    ╲    ← 飘带(左右)

圆轮廓, 无锐角
```

### 细节分解

| 部位 | 设计元素 |
|---|---|
| **头部** | 光头，额间红点(第三只眼意象)，耳垂拉长(佛像感) |
| **颈部** | 念珠项链 1 圈 |
| **上装** | 蓝色僧衣，短袖，露出手臂 |
| **手腕** | 金色护腕，念珠手链 |
| **腰部** | 金色腰带，飘带末端 |
| **下装** | 蓝色宽松裤，裤脚收紧 |
| **脚** | 草鞋或赤脚 |
| **手持** | 念珠串(闲置时)，气功拳套(战斗时) |
| **飘带** | 2 条白色飘带，从腰间飘起 |

## 动画参考

### 待机 (Idle)

```
- 轻微呼吸起伏
- 拳头握紧又放松(循环)
- 念珠轻微摇晃
- 飘带缓慢飘动
```

### 行走 (Walk)

```
- 双手握拳前后摆动
- 步幅小(腿短)
- 飘带随走动飘动
- 身体轻微上下弹跳
```

### 轻攻 (Light Attack)

```
Light 1: 左直拳 (快速, 前冲)
Light 2: 右勾拳 (接上, 身体旋转)
Light 3: 双掌推 (气功波起手)

每招: 蓄力 0.1s → 出击 0.1s → 收招 0.3s
```

### 重攻 (Heavy Attack)

```
气功波:
- 双手胸前合十 0.5s
- 蓝色气功能量聚集 0.5s
- 推出, 冲击波 0.5s
- 收招 0.5s
```

### 必杀 (Ultimate)

```
如来神掌:
- 跳起, 镜头拉近 1s
- 空中双手合十, 金光环绕 1s
- 俯冲, 巨掌从天而降 1s
- 爆炸特效, 镜头震动 1s
```

### 闪避 (Dodge)

```
气功瞬移:
- 身体化为蓝光 0.1s
- 瞬移到目标方向 0.1s
- 重新凝结 0.1s
- 残留蓝色残影
```

## 建模优先级

P0 阶段只建模核心可见元素:

1. **基础身体** (头+躯干+手臂+腿) — Day 3-4
2. **僧衣基础** (简单衣服模型) — Day 4-5
3. **关键配件** (念珠、护腕、飘带) — Day 5-6
4. **武器** (气功拳套 — 简单即可) — Day 6-7

## 骨骼适配

由于 Q版 三头身比例与 Mixamo 标准骨骼不同:

```
Mixamo 标准 vs 我们的 Q版:

头部长度:
Mixamo: 1/7.5 总高 (写实)
我们:   1/3 总高 (Q版)

调整方法:
1. Mixamo 绑定后, 在 Blender 中调整骨骼比例
2. 缩短腿部骨骼 (从 45% → 33% 总高)
3. 增大头部骨骼
4. 短缩手臂/腿的中间段
```
