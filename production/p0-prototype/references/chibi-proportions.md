## 各职业差异

### 气功小武僧

```
特征: 圆壮, 光头, 飘带
- 头: 光头, 略扁圆, 额间红点
- 躯干: 壮实 (比标准宽 10%)
- 手臂: 粗, 戴气功拳套
- 腿: 粗短
- 整体: 圆轮廓, 无锐角
```

### 糖果魔导师

```
特征: 纤细, 尖帽, 大杖
- 头: 正常 Q版, 戴尖帽
- 躯干: 纤细 (比标准窄 5%)
- 手臂: 细, 袖口宽大
- 腿: 细, 穿长裙/长靴
- 整体: 三角轮廓 (上小下大)
```

### 萌鬼狂战士

```
特征: 壮硕, 鬼角, 巨锤
- 头: 略大, 有鬼角
- 躯干: 最壮 (比标准宽 20%)
- 手臂: 最粗, 护腕
- 腿: 粗, 重靴
- 整体: 方轮廓, 下盘稳
```

## 面部特征 (通用 Q版)

```
        眉毛 (短而浓)
      ──────────────
    ┌────────────────────┐
    │  👁              👁  │  大眼 (占脸宽 1/3)
    │  ●               ● │  瞳孔
    └────────────────────┘
           \    /
            \  /
             ‿  ‿        小嘴 (微笑)
```

**关键比例**:
- 眼睛: 占脸宽 1/3，位于脸部中间偏下
- 鼻子: 极小（点或无）
- 嘴巴: 小，位于下巴上方 1/4 处
- 眉毛: 短而 expressive

## 建模检查清单

- [ ] 总高 120cm (3 头高)

- [ ] 眼大 (占脸宽 1/3)
- [ ] 轮廓圆润无锐角
- [ ] 手脚短小可爱
- [ ] 职业特征明显 (武僧飘带/魔导师帽子/狂战士鬼角)
- [ ] T-Pose 正确 (用于 Mixamo 绑定)
