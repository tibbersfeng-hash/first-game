# P0 原型 — 每周任务清单

> **角色**: 气功小武僧 (Huikong)
> **比例**: 7.5 头身（正常比例，动漫风格化）
> **风格**: 参考崩坏3 / 原神 / 绝区零
> **周期**: 3 周 (Day 1-21)
> **工具链**: tccli ai3d (AI 3D生成) + Blender (修整) + 混元绑骨 (自动绑骨+动画)

---

## Week 1: AI 3D 生成 + Blender 修整 + Block-out

### Day 1 (环境安装)

- [ ] 安装 Blender 4.x
- [ ] 配置 Blender (English, Node Wrangler 插件)
- [ ] 确认 tccli ai3d 服务可用（腾讯云混元 3D）
- [ ] 通读本指南
- [ ] 阅读 `references/standard-proportions.md` (7.5 头身比例)
- [ ] 阅读 `references/huikong-design.md` (武僧设计)

### Day 2 (AI 3D 生成探索)

- [ ] 使用 `tools/generate_3d_models.py` 生成武僧草稿
- [ ] 尝试不同 prompt 变体（5 个方向）:
  - 写实动漫风（类原神）
  - 风格化动漫风（类崩坏3）
  - 半写实风（类绝区零）
  - 更壮硕版本
  - 更修长版本
- [ ] 下载最接近的 1-2 个，导入 Blender 分析
- [ ] 检查比例是否符合 7.5 头身

### Day 3 (比例校准)

- [ ] 在 Blender 中创建比例参考方块 (185cm 总高, 25cm 头高)
- [ ] 调整 AI 生成模型到准确 7.5 头身比例
- [ ] 检查身体各部位比例:
  - 腿部 ≥ 全身 1/2
  - 手臂比例正确
  - 肩宽与头宽比例
- [ ] 练习: Blender 基础编辑 (G/R/S 移动/旋转/缩放)

### Day 4 (模型修整 - 身体)

- [ ] 修整头部: 光头造型 + 额间红点 (建模或后期材质)
- [ ] 修整面部: 动漫风格化 (大眼/小鼻/正常嘴)
- [ ] 调整体型: 修长有力（武僧精干型）
- [ ] 检查肌肉线条轮廓
- [ ] 保存版本 (huikong_body_v01.blend)

### Day 5 (模型修整 - 服装)

- [ ] 修整僧衣: 无袖/短袖 + 斜襟交领
- [ ] 修整裤子: 白色宽松武术裤
- [ ] 添加飘带: 2 条橙金色，从腰带飘出
- [ ] 添加护腕: 金色护腕 + 前臂绑带
- [ ] 检查衣服与身体的穿插
- [ ] 保存版本 (huikong_body_v02.blend)

### Day 6 (细节 + 武器)

- [ ] 添加手套: 露指气功手套
- [ ] 添加靴子: 深棕长靴
- [ ] 添加腰带: 金橙宽腰带
- [ ] 气功拳套 (简单模型即可)
- [ ] 细节检查: 额间红点、镶边等
- [ ] 保存版本 (huikong_body_v03.blend)

### Day 7 (拓扑 + 导出测试)

- [ ] 检查拓扑 (四边形为主, < 50k 面)
- [ ] 应用 Subdivision Surface (如需要)
- [ ] T-Pose 最终检查
- [ ] **导出 FBX 测试** → 导入 UE5 检查尺寸和比例
- [ ] 📝 Week 1 评审记录

**Week 1 交付物**:
- ✅ 武僧 block-out 模型 FBX
- ✅ 7.5 头身比例正确 (185cm)
- ✅ UE5 导入测试通过

## Week 2: 贴图 + 混元绑骨 + 基础动画

### Day 8 (UV 展开)

- [ ] UV 标记 Seam (接缝)
- [ ] UV Unwrap
- [ ] 检查 UV 利用率和拉伸
- [ ] UV 布局整理
- [ ] 注意: 衣服和身体分开 UV 岛

### Day 9 (贴图绘制)

- [ ] 创建材质槽位 (身体/僧衣/裤子/飘带/靴子)
- [ ] Base Color 贴图 (2048×2048)
- [ ] 配色参考: `references/huikong-design.md`
- [ ] 额间红点 (红色圆点)
- [ ] 金橙镶边
- [ ] 保存版本 (huikong_textured_v01.blend)

### Day 10 (贴图完善 + 导出)

- [ ] 完善贴图细节
- [ ] 导出贴图 (PNG/TGA)
- [ ] FBX 导出设置:
  - 包含 Mesh + UV + Material
  - 不包含骨骼 (混元绑骨会加)
  - Apply Scale & Rotation
- [ ] 导出 huikong_final_mesh.fbx

### Day 11 (混元绑骨)

- [ ] 使用腾讯云混元绑骨服务
- [ ] 上传 FBX 进行自动绑骨
- [ ] 选择预设骨骼类型 (标准人体)
- [ ] 检查绑定结果:
  - 关节位置正确
  - 权重分配合理
  - 无穿插
- [ ] 下载绑骨后的 FBX
- [ ] 导入 Blender 验证

### Day 12 (基础动画 - 移动)

- [ ] 从混元预设动作中选取:
  - "idle_standing" → 待机
  - "walk_forward" → 行走
  - "run_forward" → 跑步
  - "jump" → 跳跃
- [ ] 每个动画检查:
  - 循环是否无缝
  - 比例是否正确
  - 动作是否自然
- [ ] 导入 Blender, 应用到绑定角色
- [ ] 预览效果

### Day 13 (动画微调)

- [ ] 检查正常比例下的动画效果:
  - 步幅是否合适 (腿长 → 步幅相应)
  - 手臂摆动是否协调
  - 跳跃高度是否合理
  - 飘带物理是否正确
- [ ] Blender 中调整关键帧 (如需要)
- [ ] 确保所有动画循环无缝
- [ ] 导出 FBX (含骨骼 + 动画)

### Day 14 (UE5 集成测试)

- [ ] 云服务器恢复后上传:
  - Rigged Character FBX
  - 所有动画 FBX
  - Base Color 贴图
- [ ] UE5 导入 SkeletalMesh
- [ ] 创建 Skeleton + Physics Asset
- [ ] 导入动画, 测试 Retarget
- [ ] 测试基础动画播放
- [ ] 📝 Week 2 评审记录

**Week 2 交付物**:
- ✅ 混元绑骨完成
- ✅ 4 个基础动画 (idle/walk/run/jump)
- ✅ UE5 导入成功, 动画可播放

## Week 3: 战斗动画 + NPR + P0 评审

### Day 15 (轻攻动画)

- [ ] 从混元预设选取或手工制作 Light Attack 1 (左直拳)
- [ ] 从混元预设选取或手工制作 Light Attack 2 (右勾拳)
- [ ] 从混元预设选取或手工制作 Light Attack 3 (双掌推)
- [ ] 每招包含: 蓄力 + 出击 + 收招
- [ ] 确保连招衔接流畅
- [ ] 参考: 李小龙/叶问 出拳节奏

### Day 16 (重攻 + 技能)

- [ ] 手工制作 Heavy Attack (气功波蓄力推出)
- [ ] 手工制作 Skill 1 (气功波远程)
- [ ] 测试 Animation Notify 触发点
- [ ] 金色气功波特效预留位置

### Day 17 (闪避 + 受击)

- [ ] 手工制作 Dodge (气功瞬移效果)
- [ ] 从混元预设选取 Hit Reaction 并修改
- [ ] Death 动画 (可从混元预设修改)
- [ ] 闪避残影效果预留

### Day 18 (UE5 Animation Blueprint)

- [ ] 创建 Animation Blueprint (ABP_Huikong)
- [ ] 实现 State Machine:
  - Locomotion (Idle/Walk/Run blend)
  - Jump (Jump → Fall → Land)
  - Attack (Light1 → Light2 → Light3)
  - Dodge
  - Hit
- [ ] 使用 BlendSpace 实现移动混合

### Day 19 (UE5 NPR 材质)

- [ ] 参考 `references/npr-reference.md`
- [ ] 创建 Material: M_Huikong_Toon
- [ ] 实现 2-tone 或 3-tone shading
- [ ] 实现 Rim Light (发丝/衣服边缘)
- [ ] 实现描边 (Inverted Hull 方法, 比 Q 版更细)
- [ ] 应用到武僧模型, 测试效果

### Day 20 (整合测试)

- [ ] 在 UE5 中测试:
  - 移动 (WASD) + 动画混合
  - 攻击 (J/K) + 动画播放
  - 闪避 (Space) + 动画播放
  - 锁定 (Tab) + 相机切换
  - NPR 渲染效果
- [ ] 修复发现的问题
- [ ] 性能检查 (> 60fps)

### Day 21 (P0 评审)

- [ ] 准备 P0 评审文档
- [ ] 录制演示视频 (30s)
- [ ] 总结:
  - 风格是否满意
  - 手感是否达标
  - 管线是否可行
  - 后续调整方向
- [ ] 📝 P0 评审报告

- ✅ 完整战斗动画集 (15 个动画)
