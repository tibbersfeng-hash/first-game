## 八、制作管线参考

### 3D 模型管线（参考慧空已验证流程）

```
概念参考图 (ref.png, 已有)
    │
    ▼
Blockout / 基础模型 (Blender / AI 生成)
    │
    ▼
高模雕刻 → Retopology 低模
    │
    ▼
UV 展开 → 贴图绘制 (Substance Painter / AI)
    │
    ▼
骨骼绑定 (Rigging) → UE5 Mannequin 标准骨骼适配
    │
    ▼
导入 UE5 (FBX) → 材质实例 → 动画绑定
```

### 3D 动画管线

```
方式 A: Mixamo / 动画库重定向 (快速原型)
方式 B: Blender 手K (精细品质)
方式 C: AI 动作生成 (Move.ai / DeepMotion)
    │
    ▼
UE5 Animation Blueprint → BlendSpace → 状态机
    │
    ▼
行为树驱动 (EnemyAIController)
```

### 2D 贴图生成管线

```
AI 生图 (wanx2.1 / qwen-image) → 序列帧
    │
    ▼
后处理: 去背景 + 统一尺寸
    │
    ▼
导入 UE5 → 配置 Flipbook / Niagara 粒子参数
```
