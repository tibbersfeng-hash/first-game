# PROTOTYPE - NOT FOR PRODUCTION
# Combat Concept - 格斗萌主

## 原型问题
简化3-4键搓招 + 命中停顿 + 屏幕震动 + 伤害数字，能否产生"Q版角色打出爆炸打击感"的反差爽？

## 操控

| 按键 | 功能 |
|------|------|
| A/D | 左右移动 |
| W | 跳跃 |
| J | 轻攻击 |
| K | 重攻击 |
| L | 特殊攻击 |

## 连招表

| 连招名 | 输入序列 | 特点 |
|--------|---------|------|
| 基础连 | J → J → K → L | 标准combo，逐步升级打击感 |
| 速连 | J → J → J | 快速轻攻收尾，容易上手 |
| 力连 | K → L | 重击起手大招收尾，高风险高回报 |

## 核心反馈系统

1. **命中停顿 (Hit-Stop)**: 轻攻3帧、重攻8帧、必杀12帧——击中瞬间画面冻结
2. **屏幕震动**: 轻攻无、重攻中等、必杀强烈
3. **伤害数字**: 白色轻攻、黄色重攻、红色必杀、金色暴击
4. **角色变形**: 攻击时挤压拉伸、命中时对手挤压
5. **连招计数器**: 实时显示 COMBO x N

## 在 Godot 中运行

1. 用 Godot 4.6+ 打开 `prototypes/combat-concept/` 目录
2. 按 F5 运行
3. 用 A/D 移动到木桩旁，开始按 J/K/L 连招

## 原型验证标准

- [ ] 2分钟内玩家自愿重试不同连招组合
- [ ] 命中停顿让打击感"有分量"
- [ ] Q版角色(色块)打出夸张特效的反差感成立
- [ ] 3键连招输入门槛足够低，不搓招也能爽
